set_stream_decimation_factor(decimation_factor: int, stream: int = 1)¶

Set the stream’s decimation factor. This function sets the decimation factor of a stream. The decimation factor defines how many samples are sent over the stream. A decimation factor of 1 means that every sample is sent, a value of 2 means every 2nd sample is send, 3 means every 3rd sample, and so on. note The decimation factor should only be increased if the stream’s data rate is too high, and packets are being dropped.

Parameters

  • decimation_factor (int) – decimation factor of a stream

  • stream (int) – ADPD Stream

Returns

returns the status of the set_stream_decimation_factor method

Return type

int

Raises
